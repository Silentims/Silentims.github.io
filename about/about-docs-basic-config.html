# 主题配置

提示

此处只演示网站首页、顶部导航栏、网站标题栏图标、网站 Logo的简单配置，进阶配置可参考官方的主题配置open in new window

在/docs/.vuepress目录下，创建public/images文件夹，以及config.js文件

# 进入.vuepress目录
cd docs/.vuepress/
# 创建存放网站图片的目录
mkdir -p public/images
# 创建 config.js 配置文件
touch config.js

修改config.js文件内容

此处直接复制下面的基础配置，再根据注释，自行修改和删减

module.exports = {
  lang: 'zh-CN',	//语言
  title: 'SW·BOOK',		//(全局默认)导航栏左上角标题
  description: '时间会回答成长，成长会回答梦想',	//(全局默认)描述

  // head 可以配置网站标题栏的图标，如果不配置默认显示 vuepress 图标
  head: [
    //支持图片 url 和本地，这里使用本地的相对路径，把准备好的 ico 图标文件复制到/images目录下
    ['link', { rel: 'icon', href: '/images/favicon.ico' }]
  ],

  //主题配置 详细可参考https://v2.vuepress.vuejs.org/zh/reference/default-theme/config.html 
  themeConfig: {

    //导航栏左侧图标： 图片需要放在 /images 目录下
    logo: '/images/favicon.ico',

    
    //导航栏右侧分类：这里的 link 路径对应是在 docs 根目录下，详细配置参考官方，这里容易配置错误
    navbar: [
      // 方式一：NavbarItem
      {
        text: '首页',
        link: '/',
      },
      // 方式二：NavbarGroup 嵌套最多 2 层
      {
        text: '分类',
        children: [
          {
            text: '子类a',
            link: '/md/sub-a'	//表示链接到/docs/md/目录下的 sub-a.md 文件，即 /docs作为根目录可省略
          },
          {
            text: '子类b',		//这里子类b包括两个2 层分类，官方表示最多支持 2 层
            children: [
              {
                text: '子类b-1',
                link: '/md/sub-b-1'  
              },
              {
                text: '子类b-2',
                link: '/md/sub-b-2'  
              }
            ]    
          }
        ]
      },
      // 方式三：字符串 - 页面文件路径（会自动转换为 NavbarItem，文件路径为 link，文件标题为 text）
      '/about.md',
    ],
    
    //其他配置...
  
  }
}

由于配置文件中涉及导航栏中的几个测试文章，所以也分别在/docs目录下临时创建几篇文章，为了演示，默认都在这几篇文章添加对应的一级标题即可:

# 进入 docs 目录下，创建一个用于存放文章的目录 md
cd docs
mkdir md
# 创建关于页
echo '# 关于' > about.md
# 创建文章页，并各自添加一个一级标题（此处也可以不通过命令行创建创建）
cd md
echo '# 子类a' > sub-a.md
echo '# 子类b-1' > sub-b-1.md
echo '# 子类b-2' > sub-b-2.md

至此，基础的配置已经完成，主要的文件结构如下

swbook	# 项目根目录
├── docs	# 网站内容根目录
│   ├── .vuepress	# vuepress 目录
│   │   ├── config.js	# 主配置文件
│   │   ├── public	# 存放公共静态资源
│   │   │   └── images	
│   │   │       ├── favicon.ico	# 网站标题栏图标
│   │   │       └── logo.png	# 网站 logo 图片
│   ├── README.md	# 首页
│   ├── about.md	# 关于页
│   └── md	# 文章目录 ps：可以自定义很多同级和子级目录，只要是在 /docs 下即可
│       ├── sub-a.md	# 文章 1
│       ├── sub-b-1.md	# 文章 2
│       └── sub-b-2.md	# 文章 3
├── package-lock.json
└── package.json	# 项目配置文件
